# Зелёные горки — рейтинг 2 смены

Готовый статический веб-интерфейс для размещения на GitHub Pages.

## Что внутри

- `index.html` — страница рейтинга
- `style.css` — оформление в стиле выбранной инфографики
- `app.js` — загрузка и редактирование данных
- `data.json` — данные рейтинга отрядов
- `assets/reference-infographic.png` — выбранный визуальный референс

## Как опубликовать на GitHub Pages

1. Создайте репозиторий, например `zelenye-gorki-rating`.
2. Загрузите в него все файлы из этой папки.
3. В GitHub откройте `Settings` → `Pages`.
4. В разделе `Build and deployment` выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Сохраните. Через 1–3 минуты появится ссылка на сайт.

## Как менять рейтинг

Самый простой способ:
1. Откройте сайт.
2. Нажмите `Редактировать данные`.
3. Измените названия отрядов, подписи и баллы.
4. Нажмите `Сохранить на этом устройстве`.

Для общего обновления на сайте:
1. Отредактируйте файл `data.json`.
2. Замените значения в блоке `ranking`.
3. Загрузите обновлённый `data.json` в GitHub.

## Пример строки рейтинга

```json
{
  "place": 1,
  "name": "3 отряд «Энергия»",
  "leader": "Вожатые: Анна и Максим",
```
