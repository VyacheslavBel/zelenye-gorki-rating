# Зелёные горки — рейтинг 2 смены

Обновлённый статический сайт для GitHub Pages.

## Что изменено

- шапка `index.html` заменена на изображение по вашему образцу;
- добавлена отдельная страница `admin.html`;
- на `admin.html` можно просто вводить значения и сохранять их;
- `index.html` показывает эти данные автоматически;
- данные сохраняются через `localStorage` внутри браузера;
- можно скачать актуальный `data.json` и загрузить его в GitHub.

## Файлы

- `index.html` — публичная страница рейтинга;
- `admin.html` — страница администрирования;
- `style.css` — общий стиль;
- `app.js` — логика index;
- `admin.js` — логика admin;
- `data.json` — стартовые данные;
- `assets/header-reference.png` — шапка-изображение.

## Как это работает

### Для локального управления
1. Откройте `admin.html`.
2. Введите данные.
3. Нажмите `Сохранить`.
4. Откройте `index.html` — там появятся изменения.

> Это работает в одном и том же браузере, потому что используется `localStorage`.

### Для публикации на GitHub Pages
1. Загрузите все файлы в репозиторий GitHub.
2. Включите GitHub Pages.
3. Чтобы обновить сайт для всех пользователей:
   - откройте `admin.html`,
   - внесите правки,
   - нажмите `Скачать data.json`,
   - замените файл `data.json` в репозитории,
   - закоммитьте изменения.

После этого обновления увидят все родители и дети.
